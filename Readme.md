# Real-Time Mask Detector

The Social Distancing Detector can use in any kind of mob places (Example – Mall, Streets, Airport, and hospitals) to detect the
peoples who don’t follow the social distancing regulations.

## Installation

Use the package manager pip to install requirements

```bash
pip install -r requirements.txt
```

## Run main.py file
File generate url [Postman link](http://127.0.0.1:5000/social_distancing_detection)
